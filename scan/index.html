EFF SSL Observatory scanning system

This is a scanning system for the Electronic Frontier Foundation's SSL Observatory project. This project works to improve the security of HTTPS/TLS/SSL services on the Web by collecting and auditing the certificates on publicly-visible HTTPS servers. EFF works with web browser manufacturers and Certificate Authorities to fix the problems the Observatory finds. You may notice brief connections from this host to port 443 on your machines with public IPv4 addresses as we collect this data.

If you have read the project page but have further questions about this machine, feel free to contact us at ssl-survey@eff.org.
